# DLDUNN
In this repository, code is for our 2021 IEEE PIMRC paper [Hybrid Precoding Design Based on Dual-Layer Deep-Unfolding Neural Network](https://ieeexplore.ieee.org/document/9569633)

## Requirements
* Python >= 3.6
* PyTorch >= 1.1.0
